[1] Успешное создание объявления пользователем, получение созданного объявления по идентификатору и получение его статистики
- Приоритет: high
- Предусловия: Авторизоваться как пользователь

Шаги:
1) Выполнить запрос POST/api/1/item - сохранить объявление
Ввести валидные данные

Пример запроса:
{
  "sellerID": 890520,
  "name": "testтест",
  "price": 900230900,
  "message": "объявление",
  "statistics": {
    "likes": 6,
    "viewCount": 1,
    "contacts": 3
  }
}
где "sellerID" - id продавца,
  "name" - название объявления,
  "price"- цена в объявлении,
  "message" - текст объявления
  "statistics" - статистика по объявлению 
    "likes" - количество добавлений объявления в избранное 
    "viewCount" - количество просмотров объявления
    "contacts" - количество контактов с продавцом, создавшим объявление

Зафиксировать id объявления из ответа

Ожидаемый результат: 
Ответ: 200 OK
 созданному объявлению присвоен id

Пример ответа:
{
    "status": "Сохранили объявление - aaa78029-04b9-4408-909e-e24aa0e9fdbc"
}
где "status" - статус объявления


2) Выполнить запрос GET/api/1/item/{id}
Ввести id, полученный в шаге 1

Пример запроса:
https://qa-internship.avito.com/api/1/item/aaa78029-04b9-4408-909e-e24aa0e9fdbc

Ожидаемый результат:  
Ответ: 200 OK
Пример ответа:
[
    {
        "createdAt": "2025-12-25 11:49:01.667341 +0300 +0300",
        "id": "aaa78029-04b9-4408-909e-e24aa0e9fdbc",
        "name": "testтест",
        "price": 900230900,
        "sellerId": 890520,
        "statistics": {
            "contacts": 3,
            "likes": 6,
            "viewCount": 1
        }
    }
]
где "createdAt" - дата и время создания объявления,
        "id"- id объявления
         "name" - название объявления,
         "price"- цена в объявлении,
         "sellerId" - id продавца,
        "statistics" - статистика по объявлению 
        "likes" - количество добавлений объявления в избранное 
        "viewCount" - количество просмотров объявления
        "contacts" - количество контактов с продавцом, создавшим объявление

3) Выполнить запрос GET/api/1/statistics/{id}
Ввести id, полученный в шаге 1 

Пример запроса:
https://qa-internship.avito.com/api/1/statistics/aaa78029-04b9-4408-909e-e24aa0e9fdbc

Ожидаемый результат:
Ответ: 200 OK
Пример ответа:
[
    {
        "contacts": 3,
        "likes": 6,
        "viewCount": 1
    }
]где  "likes" - количество добавлений объявления в избранное 
        "viewCount" - количество просмотров объявления
        "contacts" - количество контактов с продавцом, создавшим            объявление
Значения полей  "contacts", "likes", "viewCount" соответствуют введенным в шаге 1 значениям этих же полей.
